Write a function which organizes files in a directory. 
```
└── directory-before
    ├── csv.csv
    ├── image1.jpg
    ├── image2.jpg
    ├── text1.txt
    └── text2.txt
```

```
├── directory-after
│   ├── csv
│   │   └── csv.csv
│   ├── jpg
│   │   ├── image1.jpg
│   │   └── image2.jpg
│   └── txt
│       ├── text1.txt
│       └── text2.txt
```
